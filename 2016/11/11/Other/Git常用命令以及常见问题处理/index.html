  4. git cherry-pick 不太常用，了解

  5. patch简单使用
    patch的生成又多重方式git diff和git formate-patch都是命令都过于复杂灵活度不高，一般生成patch时都是使用一个图形化工具。

  6. 本地分支与远程分支进行关联
    切换分支后忘记关联远程分支，又不想将代码merge到原分支，可以将当前分支与远程分支进行关联，然后pull，push即可将修改提到服务器
