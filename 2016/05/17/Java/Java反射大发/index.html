————关于反射调用方法————–

    • Class.getMethod()此方法只能获取到public的方法

    • Class.getDeclaredMethod()此方法可以获取到所有的方法，没有修饰符的影响
      注意：从效率上讲，getDeclaredMethod()方法的效率是比getMethod()低的，因此如果知道需要获取的Method的修饰符，最好选择效率高的方法。

    } else {
